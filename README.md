# AWS VPC Deep Dive

This repository contains structured documentation of AWS Virtual Private Cloud (VPC) concepts, covering fundamentals to advanced networking and monitoring features with practical explanations and diagrams.

The goal of this project is to demonstrate a real-world understanding of AWS VPC design, security, connectivity, and traffic visibility.

---

## What This Repository Covers
- VPC fundamentals and working
- CIDR range planning and best practices
- Public and private subnet design
- Route tables, Internet Gateway, and NAT Gateway
- Network security using Security Groups and NACLs
- VPC Flow Logs for traffic monitoring and troubleshooting
- VPC Peering for inter-VPC communication
- Transit Gateway for scalable multi-VPC connectivity
- Common VPC design and configuration mistakes

Each topic is documented in a separate folder with clear explanations and architecture diagrams.
